🎓 Atividade Prática: Laravel + Git

Curso Técnico em ADS - Desenvolvimento Web com MVC

Atividade em Dupla

📋 Objetivos de Aprendizagem

  • Compreender e aplicar o padrão MVC (Model-View-Controller)
  • Utilizar o framework Laravel para desenvolvimento web
  • Implementar mapeamento objeto-relacional (ORM) com Eloquent
  • Configurar rotas e mapeamento de URLs
  • Versionar código utilizando Git e GitHub
  • Trabalhar em equipe com controle de versão

📝 Descrição da Atividade

Vocês deverão criar um Sistema de Gerenciamento de Livros (Biblioteca Digital) utilizando Laravel. O sistema permitirá cadastrar, listar, editar e excluir livros.

💡 Dica: Trabalhem em pares! Um colega cria o repositório e convida o outro como colaborador.

⚙️ Pré-requisitos

0 Instalações necessárias:
  • PHP 8.0 ou superior
  • Composer
  • Git instalado
  • Conta no GitHub
  • VS Code ou outro editor

🚀 Passo a Passo

1 Criando o Projeto Laravel

Abra o terminal e execute:

composer create-project laravel/laravel biblioteca
cd biblioteca
2 Configurando o Banco de Dados

Edite o arquivo .env:

DB_CONNECTION=sqlite
# Ou use MySQL:
# DB_CONNECTION=mysql
# DB_DATABASE=biblioteca
# DB_USERNAME=root
# DB_PASSWORD=

Para SQLite, crie o arquivo:

touch database/database.sqlite
3 Criando o Model e Migration

Execute no terminal:

php artisan make:model Book -mcr
-m = cria migration
-c = cria controller
-r = controller com recursos (CRUD completo)
4 Configurando a Migration

Edite o arquivo em database/migrations/xxxx_create_books_table.php:

<?php

use Illuminate\\Database\\Migrations\\Migration;
use Illuminate\\Database\\Schema\\Blueprint;
use Illuminate\\Support\\Facades\\Schema;

return new class extends Migration
{
    public function up(): void
    {
        Schema::create('books', function (Blueprint $table) {
            $table->id();
            $table->string('title');
            $table->string('author');
            $table->string('isbn')->unique();
            $table->integer('pages');
            $table->decimal('price', 8, 2);
            $table->timestamps();
        });
    }

    public function down(): void
    {
        Schema::dropIfExists('books');
    }
};

Aplique a migration:

php artisan migrate
5 Configurando o Model (ORM)

Edite app/Models/Book.php:

<?php

namespace App\\Models;

use Illuminate\\Database\\Eloquent\\Model;

class Book extends Model
{
    // Campos que podem ser preenchidos em massa
    protected $fillable = [
        'title',
        'author',
        'isbn',
        'pages',
        'price'
    ];
}
6 Configurando as Rotas

Edite routes/web.php:

<?php

use Illuminate\\Support\\Facades\\Route;
use App\\Http\\Controllers\\BookController;

// Rota principal
Route::get('/', function () {
    return view('welcome');
});

// Rotas resource para Books (CRUD completo)
Route::resource('books', BookController::class);
Rotas criadas automaticamente:
GET /books (listar) | POST /books (criar)
GET /books/create (form criar) | GET /books/{id} (mostrar)
GET /books/{id}/edit (form editar) | PUT /books/{id} (atualizar)
DELETE /books/{id} (excluir)
7 Controller - Métodos CRUD

Edite app/Http/Controllers/BookController.php:

<?php

namespace App\\Http\\Controllers;

use App\\Models\\Book;
use Illuminate\\Http\\Request;

class BookController extends Controller
{
    // Listar todos os livros
    public function index()
    {
        $books = Book::all();
        return view('books.index', compact('books'));
    }

    // Mostrar formulário de criação
    public function create()
    {
        return view('books.create');
    }

    // Salvar novo livro
    public function store(Request $request)
    {
        $request->validate([
            'title' => 'required|max:255',
            'author' => 'required',
            'isbn' => 'required|unique:books',
            'pages' => 'required|integer',
            'price' => 'required|numeric'
        ]);

        Book::create($request->all());
        return redirect()->route('books.index')
            ->with('success', 'Livro cadastrado com sucesso!');
    }

    // Mostrar um livro específico
    public function show(Book $book)
    {
        return view('books.show', compact('book'));
    }

    // Mostrar formulário de edição
    public function edit(Book $book)
    {
        return view('books.edit', compact('book'));
    }

    // Atualizar livro
    public function update(Request $request, Book $book)
    {
        $request->validate([
            'title' => 'required|max:255',
            'author' => 'required',
            'isbn' => 'required|unique:books,isbn,'.$book->id,
            'pages' => 'required|integer',
            'price' => 'required|numeric'
        ]);

        $book->update($request->all());
        return redirect()->route('books.index')
            ->with('success', 'Livro atualizado com sucesso!');
    }

    // Excluir livro
    public function destroy(Book $book)
    {
        $book->delete();
        return redirect()->route('books.index')
            ->with('success', 'Livro excluído com sucesso!');
    }
}

🔄 Versionando com Git e GitHub

1 Inicializando o Git (Aluno 1)
git init
git add .
git commit -m "Primeiro commit - Projeto Biblioteca Laravel"

# Crie o repositório no GitHub e adicione:
git remote add origin https://github.com/SEU_USUARIO/biblioteca.git
git branch -M main
git push -u origin main
2 Adicionando Colaborador (no GitHub)
  1. Vá em Settings do repositório
  2. Clique em "Collaborators"
  3. Adicione o usuário do GitHub do seu colega
  4. Ele receberá um convite por email
3 Clonando o Repositório (Aluno 2)
git clone https://github.com/USUARIO/biblioteca.git
cd biblioteca
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
4 Fluxo de Trabalho em Equipe
# Sempre antes de começar a trabalhar:
git pull origin main

# Após fazer alterações:
git add .
git commit -m "Descrição das alterações"
git push origin main

# Em caso de conflito:
git pull origin main
# Resolva os conflitos manualmente
git add .
git commit -m "Resolvendo conflitos"
git push origin main

🛠️ Comandos Úteis do Laravel

# Criar controller
php artisan make:controller NomeController

# Criar model com migration
php artisan make:model Nome -m

# Criar migration
php artisan make:migration create_tabela_table

# Rodar migrations
php artisan migrate

# Desfazer última migration
php artisan migrate:rollback

# Limpar cache
php artisan cache:clear
php artisan config:clear
php artisan route:clear

# Listar rotas
php artisan route:list

# Modo manutenção
php artisan down
php artisan up
